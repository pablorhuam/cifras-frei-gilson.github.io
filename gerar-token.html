🔑 Gerador de Tokens de Acesso

Use esta ferramenta para gerar tokens únicos para clientes que realizaram o pagamento.

⚠️ Importante: Este arquivo deve ser mantido privado. Não compartilhe ou publique no GitHub Pages.

✅ Token Gerado com Sucesso!

📋 Passos para Liberar Acesso:

  1. Verificar pagamento no seu banco/PIX
  2. Gerar token usando este formulário
  3. Adicionar token no arquivo cifras.html (linha dos tokensValidos)
  4. Fazer commit no GitHub para atualizar o site
  5. Enviar email para o cliente com o link de acesso

🧪 Testar Token:

Após adicionar o token no cifras.html, teste o acesso:

cifras.html?token=CIFRAS-EXEMPLO-ABC123
