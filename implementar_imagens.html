🚀 Passos para Implementação

  1. Criar pastas:
    mkdir assets/gallery assets/avatars
  2. Baixar imagens: Clique nos links "Ver no Unsplash" acima e baixe as imagens
  3. Renomear arquivos: Use os nomes sugeridos (ex: maria-silva.jpg)
  4. Otimizar imagens: Redimensione para os tamanhos corretos
  5. Atualizar HTML: Substitua os placeholders pelo código fornecido
  6. Testar: Verifique se todas as imagens carregam corretamente
